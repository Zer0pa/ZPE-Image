# zpe-image-codec

Bounded sparse-stroke image codec split out of the IMC monorepo.

This repo owns the Geogram 4 `L2` bounded release surface:

- sparse-stroke geometry-positive images only
- hard reject buckets for mixed and natural images
- live authority router parity against the bounded scope

This repo does not claim a unified general-image codec.

Install `zpe-core` first in the same environment, then install this repo and run the standalone authority test and matrix rerun.
