# Bounded Scope

## Primary Route

The primary route in this repo is for sparse-stroke figures with clear figure-ground separation. On the fresh verification pack, that route accepts:

- `glyph_a`
- `fork_tree`
- `loop_spine`
- `maze_turns`
- `serpentine`

## Narrower Route

The repo also retains a narrower route for hole-bearing sparse figures. It accepts:

- `glyph_a`
- `loop_spine`
- `maze_turns`

It does not widen the primary sparse-stroke claim and it is not the default route.

## Hard Boundaries

This repo keeps the following outside scope:

- filled-mass mixed images
- textured nuisance images
- smooth gradients and broad natural-image style content
- any claim that bounded-pack acceptance equals general coverage

## Summary

ZPE-Image is a bounded image encoder for two explicitly limited scopes. Broad natural-image coverage remains out of scope.
